a simple DNS lookup tool
GitHub repository
Ever wondered what exactly is going on behind the scenes when you make a DNS query? This tool shows you how it works!
We'll start by querying one of the root DNS servers (192.203.230.10) and in 3 steps, get to an IP address for the domain.
It just looks up A records (since the goal is just to show how it works), try the main DNS lookup tool to make other kinds of queries.
